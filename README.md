<h1><b>AI-Student-Chatbot</b></h1>
<h3>An AI powered chatbot structured to support the students with their daily lessons, inquiry and assist by providing guidance. This project in developed with GROQ and designed by Gradio giving a web-based interface.</h3>

<h2><u><b>🔷FEATURES</b></u></h2>
•	AI powered responses<br>
•	Interactive chats<br>
•	Lightweight and swift handling<br>
• User friendly interface 

<h2><u><b>🔷TECHNOLOGY USED</b></u></h2>
•	Python<br>
•	Gradio (Frontend UI)<br>
•	Groq
